# FormCheck-AI

A machine learning system that classifies squat form using pose estimation.

## Tech Stack
- Python
- MediaPipe (Pose Estimation)
- OpenCV
- scikit-learn
- Jupyter

## Project Structure
- `data/` — raw datasets (not tracked)
- `notebooks/` — exploration and analysis
- `src/` —  source code
- `models/` — trained model files
