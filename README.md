# FormCheck-AI

A machine learning system that analyzes push up form and classifies it as correct or incorrect using pose estimation (MediaPipe).

## What it does
Takes a push up video as input, extracts body joint coordinates using MediaPipe, computes biomechanical angles, and classifies whether the form is correct or incorrect.

## Tech Stack
- Python
- MediaPipe (Pose Estimation)
- OpenCV
- scikit-learn
- Jupyter

## Project Structure
- `data/` — raw datasets (not tracked)
- `notebooks/` — exploration and analysis
- `src/` — reusable source code
- `models/` — trained model files
